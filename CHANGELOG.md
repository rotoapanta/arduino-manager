# Changelog

## [1.2.0] - 2026-03-31

### Added
- Automatic icon extraction from AppImage
- Desktop database update
- Icon cache refresh
- Wrapper execution with `--no-sandbox`

### Improved
- Better installation flow
- Improved logging system
- Cleaner uninstall process

### Fixed
- Icon not appearing in GNOME
- Execution issues on Ubuntu 24.04

---

## [1.1.0] - 2026-03-25

### Added
- Interactive menu mode
- CLI mode (`--install`, `--remove`, `--update`)
- Installation from local file and URL
- Logging system

### Added
- Initial release
- Basic AppImage installation
